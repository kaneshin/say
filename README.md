# say

say - Convert text to audible speech

## Usage

```
$ say べんぞうさんの中あったかいなりー
```

## Install

```
$ go get github.com/mattn/say
```

## Setup

1. Open https://cloud.voicetext.jp/ in your browser.
2. Get API key.
3. Paste API key into `~/.voicetext-apikey`
